# NOTA IMPORTANTE

Per completare la sezione statuto, dovrai:

1. **Caricare il PDF dello statuto**: 
   - Salva il file PDF dello statuto come "statuto-auvr.pdf" nella cartella public/
   
2. **Aggiornare i dati dell'associazione** nel file StatutoPage.tsx:
   - Codice Fiscale
   - Partita IVA (se applicabile)
   - Sede legale completa
   - Data di costituzione
   - Nomi degli organi sociali
   - Contatti amministrativi
   
3. **Altri documenti da aggiungere nella cartella public/**:
   - regolamento-interno.pdf
   - codice-etico.pdf
   - privacy-policy.pdf
   - atto-costitutivo.pdf

4. **Creare eventualmente una pagina bilanci** se necessario

La pagina è già completamente funzionale e accessibile dal menu "Statuto".
