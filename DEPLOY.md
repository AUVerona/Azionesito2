# Deploy GitHub Pages

## Passi da seguire:

1. **Vai su GitHub**: https://github.com/AUVerona/Azionesito2
2. **Settings** > **Pages**
3. **Source**: Deploy from a branch
4. **Branch**: main
5. **Folder**: / (root)
6. **Save**

## Il sito sarà disponibile a:
https://auverona.github.io/Azionesito2/

- ✅ Base path configurato per `/Azionesito2/`
